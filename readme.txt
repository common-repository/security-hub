== Description ==

WP Security Hub is a thesis project 2022 for IED Milano - Media Design course. 
The owners are Federica Ferletti and Silvia Maiocchi.
Our idea is to create a tool useful for protecting the security of websites that are created with WordPress.
Our focus is to make an user-friendly plugin that everybody can use easily.

== Installation ==

1. Install using the WordPress built-in Plugin installer.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Plugin installation takes only the time to have a coffee, or less.
